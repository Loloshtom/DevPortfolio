# Developer Portfolio

# Opis projektu
Jest to nowoczesna, responsywna strona portfolio stworzona dla Front-end Developera. Strona zawiera sekcje: Hero, O mnie, Projekty (z wykorzystaniem CSS Grid) oraz w pełni funkcjonalny formularz kontaktowy z walidacją. Projekt został przygotowany z dbałością o semantykę HTML5 oraz dostępność (WCAG).

# Czego się nauczyłem
Podczas tworzenia tego projektu utrwaliłem i zdobyłem następujące umiejętności:
Semantyka i SEO: Wykorzystanie odpowiednich tagów HTML5 (`header`, `main`, `section`,`article`), co poprawia indeksowanie strony.
CSS Layouts: Praktyczne zastosowanie Flexboxa do nawigacji oraz CSS Grid do stworzenia responsywnej siatki projektów.
Dostępność (A11y): Implementacja atrybutów `aria-label` oraz `aria-describedby` w formularzu kontaktowym, aby strona była przyjazna dla czytników ekranu.
SVG: integracja grafik wektorowych bezpośrednio w kodzie HTML.

# Z czym miałem trudność
Walidacja formularza: Największym wyzwaniem było stworzenie czytelnego systemu powiadomień o błędach, który pojawia się dynamicznie (JavaScript) i jest zrozumiały dla użytkownika.

Responsywność: Dopasowanie siatki projektów tak, aby wyglądała estetycznie zarówno na dużych monitorach, jak i na wąskich ekranach telefonów przy zachowaniu odpowiednich odstępów.
